# Видео для лендинга

## Структура

```
videos/
├── phone/        ← видео для мокапа телефона (вертикальное 9:16)
│   ├── hero.mp4          ← главная секция Hero
│   ├── hero.webm         ← WebM-версия для Chrome
│   └── hero-poster.jpg   ← постер-кадр (первый кадр)
│
├── notebook/     ← видео для мокапа ноутбука (горизонтальное 16:10)
│   ├── crm.mp4           ← демо CRM
│   ├── crm.webm
│   └── crm-poster.jpg
│
└── README.md
```

## Требования к видео

### Телефон (phone/)
- **Разрешение:** 1080×1920 (9:16 вертикальное)
- **Формат:** MP4 (H.264) + WebM (VP9) для кросс-браузерности
- **Длительность:** 10–30 секунд, зацикленное
- **Без звука** (autoplay требует muted)
- **Битрейт:** 2–4 Mbps (баланс качество/размер)

### Ноутбук (notebook/)
- **Разрешение:** 1920×1200 (16:10 горизонтальное)
- **Формат:** MP4 (H.264) + WebM (VP9)
- **Длительность:** 10–30 секунд, зацикленное
- **Без звука**
- **Битрейт:** 3–5 Mbps

## Конвертация через ffmpeg

```bash
# MP4 (H.264, без звука, оптимизированный для web)
ffmpeg -i input.mov -c:v libx264 -crf 23 -preset slow -an -movflags +faststart output.mp4

# WebM (VP9, без звука)
ffmpeg -i input.mov -c:v libvpx-vp9 -crf 30 -b:v 0 -an output.webm

# Постер (первый кадр)
ffmpeg -i input.mp4 -vframes 1 -q:v 2 output-poster.jpg
```
